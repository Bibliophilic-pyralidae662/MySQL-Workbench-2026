## 🖥️ First Launch And Setup

When you open the application for the first time, you see the home dashboard. This screen lists your active connections. If you have no connections yet, follow this process to create one.

1.  Click the plus sign next to the MySQL Connections header.
2.  Type a name for your connection in the Connection Name box.
3.  Leave the Hostname as 127.0.0.1 if your database runs on your own computer.
4.  Enter the correct Port number (usually 3306).
5.  Enter your database Username.
6.  Click the Test Connection button to verify your settings.
7.  Type your password when the prompt appears.
8.  Click OK to save the connection.

The application saves these details so you do not need to enter them every time. You can create multiple connections to different database servers.

## 📊 Working With Data

The main window shows your databases in a list on the left side. Expand these items to see your tables. You can interact with your information using several methods.

### Viewing Table Content
Double-click a table name to see the data inside. The application opens a new tab showing a grid of your rows and columns. You can edit this information directly by clicking on a cell, typing new data, and pressing the Apply button.

### Creating Diagrams
The visual diagram tool creates a map of your database. Right-click your database name and select Database and then Reverse Engineer. Follow the prompts to generate a visual chart of your tables and how they link together. This tool makes it easy to understand the structure of your data.

### Querying Data
The SQL Editor allows you to run specific commands. Click the New Query Tab button to open a blank page. Type your SQL commands here and click the lightning bolt icon to run them. The results appear in a panel at the bottom of the screen.

## ⚙️ Settings And Configuration

You can customize the look and feel of the application in the Edit menu. Under Preferences, you can change the font size, the color theme, and the behavior of the SQL editor. These changes take effect after you restart the program.

If you connect to remote servers, verify that your firewall allows traffic on port 3306. If the connection fails, disable your antivirus software temporarily to see if it blocks the link.

## 🛡️ Security Best Practices

Database security keeps your information safe from outside threats. Follow these guidelines while using the software.

*   Always use a strong password for your database user accounts.
*   Do not store passwords in plain text files on your computer.
*   Back up your database regularly using the Data Export tool found in the menu.
*   Only provide read-only access to users who do not need to change data.

The software includes tools to help you manage users and their level of access. Access the Users and Privileges tab to view or change these settings.

## 💡 Troubleshooting Common Issues

If the application does not start, ensure you have the latest Windows updates installed. Corrupt files sometimes cause startup errors. If you experience this, uninstall the application via your Windows Settings menu and reinstall it from the download page.

If the connection times out:
1.  Check that your database server is running.
2.  Confirm that the hostname and port number are correct.
3.  Check if your local network connection is active.
4.  Remove active connections and recreate them to clear potential errors.

For most users, the default settings work without extra configuration. Use the Help menu to access the documentation if you need further details on specific features.
